# Claude Code Intro Video Script

**Audience:** Designers | **~2-3 min** | **Casual tone**

---

- Hook: you've seen devs shipping with Claude Code, is it actually useful for designers?
- What it is: CLI tool from Anthropic, you type `claude` in your terminal and talk to it in plain English — it reads your files, writes code, builds things
- Why designers should care: go from idea to working prototype without waiting on anyone
- Demo: open terminal, type `claude`, describe a landing page (hero, CTA, feature cards), watch it build the whole thing
- Iterate live: ask it to tweak colors, spacing, hover animations — just keep talking to it
- The punchline: it's like a dev who never pushes back on your redlines
- Close: you don't need to know how to code, you just need to know what you want
