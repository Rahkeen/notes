# Claude Code Intro Video Script

**Audience:** Designers
**Tone:** Casual, conversational
**Length:** ~2-3 minutes
**Format:** What it is + quick demo

---

## HOOK (0:00 - 0:15)

So you're a designer and you've probably heard people talk about Claude Code. Maybe you've seen developers on Twitter shipping entire apps with it. And you're thinking... cool, but is this actually useful for me?

Short answer: yes. Let me show you.

---

## WHAT IS IT (0:15 - 0:45)

Claude Code is a command line tool from Anthropic. You open your terminal, type `claude`, and now you're talking to Claude — but it can actually see your files, edit your code, and build things for you.

Think of it like pair programming with an AI, except you don't need to be a programmer. You describe what you want in plain English, and it writes the code, creates the files, and runs things for you.

For designers, this is a superpower. You can go from idea to working prototype without waiting on anyone.
